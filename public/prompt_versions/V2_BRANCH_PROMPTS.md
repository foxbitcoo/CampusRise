# V2 双分支优化版

## 说明

这一版开始把原始口语化模板拆成两个明确分支：

1. 应届毕业版
2. 校友回望版

优点是比 V1 更聚焦；缺点是仍然默认用户先提供较多信息。

---

## 应届毕业版

根据【人物肖像图片】自动生成一张收藏版史诗叙事毕业海报，主题为“应届毕业”。以人物的巨大侧脸剪影作为主轮廓，剪影内部自然生长出完整且高度统一的校园叙事世界，将【学校名称】作为视觉识别核心，并围绕毕业时刻、成长终点、青春完成时、离校前夜的情绪展开画面组织。

如果提供了【专业 / 学院】，则让场景、象征符号、建筑细节、道具线索与该专业气质强绑定；如果提供了【毕业年份】，则让时间感和年代提示自然融入画面；如果提供了【个人记忆关键词】与【校园专属经历】，则优先把这些内容转化为一眼可识别但不杂乱的叙事节点，例如图书馆、操场、宿舍、实验室、社团活动、毕业典礼、晚霞中的教学楼、夜晚自习灯光、校门口、学位帽、纪念合影等。

整体不是普通拼贴，而是高级的剪影轮廓填充式叙事合成，带有双重曝光联想，但更偏向电影海报与梦幻水彩插画融合质感。画面拥有柔和空气透视、轻雾化朦胧过渡、复古纸张颗粒肌理、边缘飞白质感、手绘刷痕、大面积留白和克制高级的版式设计。

整体氛围应安静、宏大、神圣、怀旧、诗意，但底层情绪更偏向“即将告别校园的高光时刻”，带有青春、完成、纪念、珍藏的意味。所有色彩、场景、材质都必须服务于【学校名称】与人物自身气质，不要模板化背景，不要廉价奇幻素材，不要生硬拼贴，不要无关元素堆砌。

画面中若出现文字，统一使用英文。若提供【英文名】，则在画面合适位置加入一处低调精致、尺寸克制的英文手写签名【英文名】；若未提供，则不强行加入签名。

最终输出应是一张具有母校识别度、个人成长线和毕业仪式感的高完成度纪念海报。

---

## 校友回望版

根据【人物肖像图片】自动生成一张收藏版史诗叙事校友回望海报。以人物的巨大侧脸剪影作为主轮廓，剪影内部自然生长出一整套围绕【学校名称】展开的母校记忆世界，不仅体现校园空间本身，还要体现“毕业多年后回望母校”的时间厚度、身份归属感和精神传承感。

画面核心不只是“毕业瞬间”，而是“我与这所学校之间持续存在的关系”。因此画面内部应自然融合校园地标、求学记忆、时间流动、校史痕迹、代表性建筑、象征符号与成长节点。如果提供了【毕业年份】，要让年代感、时间距离和回望感更自然；如果提供了【专业 / 学院】，要让叙事更贴近人物当年的学习路径；如果提供了【个人记忆关键词】或【校园专属经历】，要把它们转化为具有回忆触发力的细节场景。

如果提供了【知名校友 / 校友精神线索】，不要简单把人物头像拼贴进画面，而是优先转译成更高级的精神关联表达。可以通过领域象征、时代符号、代表性成果意象、校史脉络或价值关键词来建立“我与这所学校精神谱系的关系”，让人物仿佛站在母校长期传承的一条河流中，而不是停留在某一次毕业典礼。

整体依旧采用高级剪影填充式叙事合成，带有双重曝光视觉联想，但风格更偏“电影海报 + 梦幻水彩插画 + 纪念典藏感”。使用柔和空气透视、轻雾化过渡、复古纸张颗粒、边缘飞白、手绘刷痕、大面积留白和克制版式。整体氛围安静、宏大、神圣、怀旧、诗意，同时比应届毕业版更强调沉淀、回望、传承、身份认同与时间纵深。

所有色彩、材质、建筑、符号和细节都必须紧密服务于【学校名称】与人物的校友身份，不要模板化校园背景，不要廉价奇幻元素，不要把校友主题做成简单人物堆砌，也不要出现与学校无关的空泛史诗素材。

画面中若出现文字，统一使用英文。若提供【英文名】，则在合适位置加入一处低调精致、尺寸克制的英文手写签名【英文名】；若未提供，则不强行加入签名。

最终输出应是一张兼具个人肖像、母校记忆、校史纵深与校友精神气质的高完成度纪念海报。
