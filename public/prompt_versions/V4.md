## 写真增强工作流版

### 说明

这是原 `Latest` 版本重命名后的版本。V4 的方向不是浮夸爆款海报，而是更偏真人写真、干净底色和高级修饰的人像叙事图。

V4 不能退化成普通人像写真。它必须保留清晰的学校身份层：学校名称、毕业年份、校训或学校精神短句、学校代表色，以及至少一个低风险学校视觉元素，例如远景地标、校徽小章、标题旁印章、页脚建筑线稿、右下角校门轮廓或校园建筑剪影。区别只是这些元素更克制、更像学校画册版式，而不是抢走人物主体。

核心变化：

1. 先看图片判断人物阶段
2. 再看图片能不能识别学校
3. 学校名称必须向用户确认
4. 学校确认后检索学校地标、校训和正向校友线索
5. 不强制要求用户先准备正脸和侧脸
6. 多张人物图用于建立更完整的人物身份模型，一张图也要尽量还原人物辨识度
7. 根据人物阶段自动选择毕业纪念方向或校友回望方向
8. 补充可选参数后直接生成，生成后再说明使用了哪些学校元素

## 工作流前置规则

### 1. 人物阶段判断

先根据图片做简单的人物阶段判断，用于决定画面方向。对用户不要说内部版本名。应该说：

- “我判断您更像是刚毕业或即将毕业的同学。”
- “我判断您更像是已经毕业比较久的同学。”

如果用户后续说明自己属于特殊情况，例如博士应届，可人工修正分支。

### 2. 人物图要求

- 不强制用户必须同时上传正脸和侧脸
- 如果用户上传多张同一人物照片，必须综合正脸、侧脸、3/4 侧脸、全身、环境图来建立人物身份模型，而不是只复制其中一张照片
- 多图输入的目标是更准确地还原原主人的五官比例、脸型、发型、气质、年龄阶段、性别呈现、常见表情和姿态可能性
- 生成结果不需要保留原照片里的固定姿势，可以在身份不漂移的前提下设计更自然、更丰富、更适合海报的表情和姿态
- 必须让熟人一眼能认出这是原来的主人；可以适度美化、增加面部细节和光影层次，但不能磨成陌生人，也不能假到像塑料脸或虚构角色
- 面部质感应介于真实写真和精致绘画之间：比普通照片更有细节、更干净、更有表达，但不能失真、不能过度美颜、不能变成夸张网红脸
- 用户只上传一张图时，也要尽量还原人物的五官、发型、眼镜、脸型、气质和辨识度
- 如果图中有侧脸或 3/4 侧脸，则优先拿来强化剪影轮廓
- 人物侧脸是主视觉，清晰度和不透明度必须高于背景校园元素
- 双重曝光只能用于轮廓内部叙事和边缘过渡，不能把人物脸部压得过淡或看不清
- 人脸五官、头发、眼镜、脸型等辨识特征必须第一眼可见
- 必须保留上传人物的性别呈现、年龄阶段和核心人物特征，不能把女性生成成男性，也不能把男性生成成女性
- 画面只允许一个主要人物表达：大侧脸肖像 / 剪影作为背景主结构；不要额外生成正面全身真人主体，除非用户明确要求
- 如果出现手、手臂、证书、书本或学位服，必须保持肢体稳定，不能出现多手、多指、断手、融合手指或不合理握持

### 3. 毕业年份规则

- 如果用户提供了毕业年份，必须严格使用用户提供的年份
- 如果用户没有提供毕业年份，且判断为刚毕业或即将毕业，必须读取当前时区下的年份作为默认毕业年份
- 如果判断为毕业多年的校友，用户没有提供毕业年份时，不要编造具体年份，只表达“毕业多年”“回望母校”
- 如果画面出现年份，所有年份文字都必须与 `毕业年份` 一致，不允许模型自由生成其他年份

### 4. 学校确认

优先从图片中自动识别：

- 校徽
- 校服 logo
- 校名文字
- 建筑门牌
- 毕业典礼背景板
- 横幅、纪念墙、校园标识

如果可靠识别到了学校，也必须向用户确认：

“我从图片里看到的学校线索像是【学校名称】，请确认这是你的学校吗？”

如果识别不到，再补问用户学校名称。

### 5. 学校检索

学校确认后，必须检索：

- 学校代表性视觉元素
- 校训或学校精神
- 学校代表色、校徽色或校园代表色
- 正向知名校友线索

要求：

- 优先使用学校官网、百科、权威媒体或可靠资料
- 不确定的信息不要使用
- 不要编造校训、地标或校友
- 校友只作为精神线索或领域象征，不做生硬人物堆砌
- 生成结果返回时说明引用了哪些学校元素、色调、校训或校友参考

### 6. 学校语言规则

- 中国大陆高校：学校名称统一用简体中文
- 香港 / 澳门 / 台湾高校：使用当地常用中文写法；没有更明确要求时，优先使用繁体中文
- 海外高校：学校名称统一用英文
- 当用户只说“清华大学”时，默认指北京的清华大学，生成文字使用简体中文“清华大学”
- 只有用户明确说“台湾清华大学”“新竹清华大学”或相关表达时，才按台湾高校处理为繁体中文

## 毕业纪念方向

根据【人物肖像图片】自动生成一张收藏版史诗叙事毕业海报，主题为“刚毕业或即将毕业”。先以图中人物真实外观作为唯一人物基底，尽量还原其年龄感、性别呈现、五官、脸型、发型、眼镜特征、整体气质与辨识度；如果图中存在侧脸或 3/4 侧脸信息，则优先吸收进人物轮廓设计。如果只有一张正面或生活照，也要在不失真的前提下推导出可信的侧脸剪影轮廓。不得把女性人物生成成男性，也不得把男性人物生成成女性。

以巨大的人物侧脸剪影作为主轮廓和背景主视觉，剪影内部自然生长出完整且高度统一的校园叙事世界，将【已确认学校名称】作为视觉识别核心，并围绕毕业时刻、成长终点、青春完成时、离校前夜的情绪展开画面组织。画面中只允许有一个主要人物表达：这个大侧脸肖像 / 剪影。不要额外生成一个正面全身真人毕业生作为前景主体，不要把原图里的学位服全身照直接复制到画面中央；如需表现毕业仪式，只能用小尺度剪影、人群远景或象征性道具。若用户没有提供毕业年份，必须读取当前时区年份作为默认毕业年份；若画面出现年份，必须写【毕业年份】且所有年份一致。

必须使用学校检索结果中的强识别元素，例如校门、湖泊、主楼、图书馆、钟楼、校徽、校园轴线、代表雕塑、代表性建筑或校园空间。优先选择用户一眼能认出“这是我的学校”的视觉元素，而不是泛化的校园或毕业典礼背景。

必须结合学校校训或学校精神提取精神关键词，并把这些精神关键词转化为画面情绪、构图节奏和象征元素。例如校训强调自强、厚德、求是、创新、自由、责任、服务等，就要把这些词转译为积极、昂扬、向上的视觉语言。

整体色调应参考学校代表色、校徽色或校园代表色。优先把这些颜色作为主色或点缀色，让用户感到更接近自己的学校；同时保持高级、克制和人物清晰度，不要为了套用校色而牺牲人像辨识度。

V4 的学校身份层必须可见但克制：画面应在标题区或左侧留白区出现【已确认学校名称】；如果是刚毕业或即将毕业方向，应出现【毕业年份】；应出现一组短校训或学校精神关键词；底部、右下角、标题旁或侧边栏应出现一个低风险学校符号，例如校徽小章、校门线稿、代表建筑线稿、建筑剪影或校园色块。不要把 Latest 做成没有学校名、没有校训、没有 logo/印章/建筑线索的普通写真。

如果提供了【专业 / 学院】，则让场景、象征符号、建筑细节、道具线索与该专业气质强绑定。如果提供了【个人特质 / 希望呈现的气质】，则优先将其转化为人物姿态、光线气质和内部叙事关键词。如果提供了【校园经历 / 社团经历】，则将其转化为一眼可识别但不杂乱的叙事节点。

如果画面中出现手、手臂、证书、书本或学位服，必须保持人体解剖稳定：只能出现自然数量的手和手指，手指不能融合、断裂、重复或多出；握持证书或道具的姿势必须物理合理。除非用户明确要求，不要把复杂手部作为画面重点，优先减少手部和道具细节以提升稳定性。

整体不是普通拼贴，而是高级的剪影轮廓叙事合成，更偏向真实人物校园写真、学校画册封面与纪念海报之间的平衡。允许有限度的轮廓内叙事融合，但不能出现梦幻水彩、魔法感、神话式云海或奇幻概念海报效果。人物侧脸必须是第一视觉焦点和画面主结构，面部不透明度和清晰度高于背景校园元素；背景可以透过人物轮廓呈现，但不能让五官、头发、眼镜和脸型变糊或消失。

整体氛围安静、宏大、神圣、怀旧、诗意，但底层情绪更偏向“即将告别校园的高光时刻”，带有青春、完成、纪念、珍藏和继续向上的意味。所有色彩、场景、材质都必须服务于【已确认学校名称】与人物自身气质，不要模板化背景，不要廉价奇幻素材，不要生硬拼贴，不要无关元素堆砌。

若画面出现学校名称，严格遵守学校语言规则。若提供了【英文名 / 签名名】，则在画面合适位置加入一处低调精致、尺寸克制的签名；未提供则不强行加入。

生成结果返回时，补充说明：

- 使用了哪些学校代表性视觉元素
- 使用了哪些校训或学校精神
- 使用了哪些学校代表色、校徽色或校园代表色
- 如果参考了知名校友，说明参考了哪些真实校友，以及只使用了哪些正向精神或领域象征
