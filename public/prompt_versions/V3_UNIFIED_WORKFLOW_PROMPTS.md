# V3 统一工作流版

## 说明

这是当前最新版本。

核心变化：

1. 先看图片判断人物年龄层
2. 再看图片能不能识别学校
3. 学校名称必须向用户确认
4. 学校确认后检索学校地标、校训和正向知名校友
5. 不强制要求用户先准备正脸和侧脸
6. 一张图也要尽量还原人物辨识度
7. 根据年龄层分流到应届毕业版或校友回望版

---

## 工作流前置规则

### 1. 年龄层分流

- `30 岁及以下`：优先走 `应届毕业版`
- `30 岁以上`：优先走 `校友回望版`

对用户不要说内部版本名。应该说：

- “我判断您更像是刚毕业或即将毕业的同学。”
- “我判断您更像是已经毕业比较久的同学。”

如果用户后续说明自己属于特殊情况，例如博士应届，可人工修正分支。

### 2. 人物图要求

- 不强制用户必须同时上传正脸和侧脸
- 用户只上传一张图时，也要尽量还原人物的五官、发型、眼镜、脸型、气质和辨识度
- 如果图中有侧脸或 3/4 侧脸，则优先拿来强化剪影轮廓
- 人物侧脸是主视觉，清晰度和不透明度必须高于背景校园元素
- 双重曝光只能用于轮廓内部叙事和边缘过渡，不能把人物脸部压得过淡或看不清
- 人脸五官、头发、眼镜、脸型等辨识特征必须第一眼可见

### 3. 毕业年份规则

- 如果用户提供了毕业年份，必须严格使用用户提供的年份
- 如果用户没有提供毕业年份，且判断为刚毕业或即将毕业，必须读取当前时区下的年份作为默认毕业年份
- 例如当前日期在 2026 年，则默认毕业年份为 2026
- 如果判断为毕业多年的校友，用户没有提供毕业年份时，不要编造具体年份，只表达“毕业多年”“回望母校”
- 如果画面出现年份，所有年份文字都必须与 `毕业年份` 一致，不允许模型自由生成其他年份

### 4. 学校确认

优先从图片中自动识别：

- 校徽
- 校服 logo
- 校名文字
- 建筑门牌
- 毕业典礼背景板
- 横幅、纪念墙、校园标识

如果可靠识别到了学校，也必须向用户确认：

“我从图片里看到的学校线索像是【学校名称】，请确认这是你的学校吗？”

如果识别不到，再补问用户学校名称。

### 5. 学校检索

学校确认后，必须检索：

- 学校代表性视觉元素
- 校训
- 正向知名校友

要求：

- 优先使用学校官网、百科、权威媒体或可靠资料
- 不确定的信息不要使用
- 不要编造校训、地标或校友
- 校友只作为精神线索或领域象征，不做生硬人物堆砌
- 生成结果返回时说明引用了哪些学校元素、校训或校友参考

### 6. 学校语言规则

- 中国大陆高校：学校名称统一用简体中文
- 香港 / 澳门 / 台湾高校：使用当地常用中文写法；没有更明确要求时，优先使用繁体中文
- 海外高校：学校名称统一用英文
- 当用户只说“清华大学”时，默认指北京的清华大学，生成文字使用简体中文“清华大学”
- 只有用户明确说“台湾清华大学”“新竹清华大学”或相关表达时，才按台湾高校处理为繁体中文

## V3 应届毕业版

根据【人物肖像图片】自动生成一张收藏版史诗叙事毕业海报，主题为“刚毕业或即将毕业”。先以图中人物真实外观作为唯一人物基底，尽量还原其年龄感、五官、脸型、发型、眼镜特征、整体气质与辨识度；如果图中存在侧脸或 3/4 侧脸信息，则优先吸收进人物轮廓设计。如果只有一张正面或生活照，也要在不失真的前提下推导出可信的侧脸剪影轮廓。

以巨大的人物侧脸剪影作为主轮廓，剪影内部自然生长出完整且高度统一的校园叙事世界，将【已确认学校名称】作为视觉识别核心，并围绕毕业时刻、成长终点、青春完成时、离校前夜的情绪展开画面组织。若用户没有提供毕业年份，必须读取当前时区年份作为默认毕业年份；若画面出现年份，必须写【毕业年份】且所有年份一致。

必须使用学校检索结果中的强识别元素，例如校门、湖泊、主楼、图书馆、钟楼、校徽、校园轴线、代表雕塑、代表性建筑或校园空间。优先选择用户一眼能认出“这是我的学校”的视觉元素，而不是泛化的校园或毕业典礼背景。

必须结合学校校训提取精神关键词，并把这些精神关键词转化为画面情绪、构图节奏和象征元素。例如校训强调自强、厚德、求是、创新、自由、责任、服务等，就要把这些词转译为积极、昂扬、向上的视觉语言。

如果提供了【专业 / 学院】，则让场景、象征符号、建筑细节、道具线索与该专业气质强绑定。专业示例包括计算机、法学、建筑、医学、新闻传播、金融、公共管理、艺术设计、自动化。

如果提供了【个人特质 / 希望呈现的气质】，则优先将其转化为人物姿态、光线气质和内部叙事关键词。特质示例包括勇敢、自由、坚定、温柔、理性、热烈、独立、探索、创造、笃定、开阔。

如果提供了【校园经历 / 社团经历】，则将其转化为一眼可识别但不杂乱的叙事节点。经历示例包括学生会、辩论队、合唱团、实验室项目、支教、创业比赛、图书馆自习、操场夜跑、毕业晚会。

整体不是普通拼贴，而是高级的剪影轮廓填充式叙事合成，带有双重曝光联想，但更偏向电影海报与梦幻水彩插画融合质感。画面拥有柔和空气透视、轻雾化朦胧过渡、复古纸张颗粒肌理、边缘飞白质感、手绘刷痕、大面积留白和克制高级的版式设计。人物侧脸必须是第一视觉焦点，面部不透明度和清晰度高于背景校园元素；背景可以透过人物轮廓呈现，但不能让五官、头发、眼镜和脸型变糊或消失。

整体氛围安静、宏大、神圣、怀旧、诗意，但底层情绪更偏向“即将告别校园的高光时刻”，带有青春、完成、纪念、珍藏和继续向上的意味。所有色彩、场景、材质都必须服务于【已确认学校名称】与人物自身气质，不要模板化背景，不要廉价奇幻素材，不要生硬拼贴，不要无关元素堆砌。

若画面出现学校名称，严格遵守学校语言规则。若提供了【英文名 / 签名名】，则在画面合适位置加入一处低调精致、尺寸克制的签名；未提供则不强行加入。

最终输出应是一张具有母校识别度、个人成长线和毕业仪式感的高完成度纪念海报。

## V3 校友回望版

根据【人物肖像图片】自动生成一张收藏版史诗叙事校友回望海报。先以图中人物真实外观作为唯一人物基底，尽量还原其年龄感、五官、脸型、发型和成熟阶段的精神状态；如果图中存在侧脸或 3/4 侧脸信息，则优先吸收进人物剪影轮廓。如果只有一张正面或生活照，也要在不失真的前提下推导出可信的侧脸剪影轮廓。

以巨大的人物侧脸剪影作为主轮廓，剪影内部自然生长出一整套围绕【已确认学校名称】展开的母校记忆世界，不仅体现校园空间本身，还要体现“毕业多年后回望母校”的时间厚度、身份归属感和精神传承感。若用户没有提供毕业年份，不要编造具体年份，也不要在画面里生成随机年份。

必须使用学校检索结果中的强识别元素，例如校门、湖泊、主楼、图书馆、钟楼、校徽、校园轴线、代表雕塑、代表性建筑或校园空间。优先选择用户一眼能认出“这是我的学校”的视觉元素，而不是泛化的校园或毕业典礼背景。

必须结合学校校训提取精神关键词，并把这些精神关键词转化为画面主题。例如校训强调自强、厚德、求是、创新、自由、责任、服务等，就要把这些词转译为积极、昂扬、向上的视觉语言。

如果学校历史上有真实存在、知名度较高、形象正向的校友，可以提取其代表领域或精神关键词作为画面隐性线索。不要简单把校友头像拼贴进画面，也不要让用户必须认识这个校友才能理解画面。校友参考应服务于“我与学校精神谱系的连接”，例如科学探索、公共服务、文学表达、工程创造、教育理想、社会责任。

画面核心不只是“毕业瞬间”，而是“我与这所学校之间持续存在的关系”。因此画面内部应自然融合校园地标、求学记忆、时间流动、校史痕迹、代表性建筑、象征符号与成长节点。如果提供了【专业 / 学院】，要让叙事更贴近人物当年的学习路径；如果提供了【个人特质 / 希望呈现的气质】或【校园经历 / 社团经历】，要把它们转化为具有回忆触发力的细节场景。若用户提供信息很少，则优先从人物年龄感、图片场景、学校检索结果和成熟阶段气质中补足叙事方向。

整体依旧采用高级剪影填充式叙事合成，带有双重曝光视觉联想，但风格更偏“电影海报 + 梦幻水彩插画 + 纪念典藏感”。使用柔和空气透视、轻雾化过渡、复古纸张颗粒、边缘飞白、手绘刷痕、大面积留白和克制版式。整体氛围安静、宏大、神圣、怀旧、诗意，同时更强调沉淀、回望、传承、身份认同与继续向上的力量感。人物侧脸必须是第一视觉焦点，面部不透明度和清晰度高于背景校园元素；背景可以透过人物轮廓呈现，但不能让五官、头发、眼镜和脸型变糊或消失。

所有色彩、材质、建筑、符号和细节都必须紧密服务于【已确认学校名称】与人物的校友身份，不要模板化校园背景，不要廉价奇幻元素，不要把校友主题做成简单人物堆砌，也不要出现与学校无关的空泛史诗素材。

若画面出现学校名称，严格遵守学校语言规则。若提供了【英文名 / 签名名】，则在合适位置加入一处低调精致、尺寸克制的签名；未提供则不强行加入。

最终输出应是一张兼具个人肖像、母校记忆、校史纵深与校友精神气质的高完成度纪念海报。
