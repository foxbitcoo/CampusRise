# V4.1 提示词

公开发布候选版本：`V4.1`

## 发布定位

V4.1 聚焦小红书发布场景里的“应届毕业纪念图”。默认不再走校友回望分支，除非用户明确要求校友、返校或毕业多年回望。

V4.1 的目标是：真人精修感更强，学校身份更准确，交互更短。它不是奇幻海报，也不是普通写真，而是“真人 P 图感 + 学校官方画册感 + 毕业纪念信息层”。

## 输入规则

### 必要输入

- `人物照片`：一张或多张。
- `学校名称`：必要信息。

如果用户在请求里已经明确说出学校名称，直接使用该学校，不再二次确认。

如果用户没有明确学校名称，再从图片中识别学校线索。识别到后请用户确认；识别不到时请用户补充。

### 可选增强信息

- `姓名`：中文名或英文名，可用于签名、标题旁小字或毕业纪念信息层。
- `毕业时间 / 毕业年份`：可填写年份、月份、季节或具体日期；未提供时，应届毕业方向默认当前年份。画面文字通常优先使用清晰年份，若用户提供了更具体时间且版面允许，可以作为小字信息层出现。
- `专业 / 学院`
- `校园经历 / 社团经历`
- `个人特质 / 希望呈现的气质`
- `签名名`

这些信息不是继续流程的门槛，但能让画面更贴近用户本人。

## 交互脚手架

### 用户已明确学校

不要二次确认学校。直接进入学校研究和生成。

生成后补充一句：

> 这版我先按【学校名】和照片人物气质生成了。如果你后续补充姓名、专业、校园经历、社团、毕业时间，或者想呈现的特质，比如自由、坚定、温柔、热烈，我可以继续把画面做得更像你的真实校园记忆。

### 用户未明确学校，但图片疑似识别到学校

输出：

> 我从图片里看到的学校线索像是【学校名】。
> B. 学校信息：如果这是你的学校，请直接确认；如果不是，请告诉我正确学校。
> 可选项：姓名、专业 / 学院、校园经历 / 社团经历、想呈现的个人特质、毕业时间或签名名。信息越具体，画面会越贴近你。

### 用户未明确学校，图片也无法可靠识别

输出：

> 我还没能可靠识别学校。
> B. 学校信息：请告诉我你的学校名称，这是生成学校纪念图的必要信息。
> 可选项：姓名、专业 / 学院、校园经历 / 社团经历、想呈现的个人特质、毕业时间或签名名。

## 学校研究规则

学校元素要慢一点、准一点。对用户会检查的学校建筑和关键符号，不要只靠文字描述。

研究步骤：

1. 网页搜索学校官网、官方视觉形象、校园建筑、校训、代表色。
2. 对会成为画面可见元素的建筑、校徽、校门、题字或关键符号，下载 1-3 张官方或高可信参考图。
3. 从参考图提取建筑轮廓、立面、屋顶、窗格节奏、材质、周边环境等视觉特征。
4. 如果没有可靠参考图，不要把该元素做成大而清晰的主视觉；改成小徽章、线稿、剪影、背景氛围，或直接省略。
5. 学校名、年份、校训、校徽和建筑若可读可辨，必须准确；拿不准就降级。

高保真优先级：

1. 人物像本人。
2. 学校名称、年份、校训和关键建筑准确。
3. 手部和身体结构稳定。
4. 画面丰富度。

如果发生冲突，牺牲装饰复杂度，不牺牲学校准确性和人物识别。

## V4.1 毕业纪念提示词

根据【人物照片】生成一张应届毕业纪念图。先综合所有上传的人物照片建立同一人物身份模型，不要只复制其中一张照片。需要稳定保留人物的性别呈现、年龄阶段、脸型、五官比例、发型、眼镜、肤色、表情范围、肩颈状态和整体气质。生成结果可以适度精修、改善光线和皮肤细节，但必须让熟人一眼认出是本人；不要变成陌生网红脸、塑料脸或虚构角色。

画面方向是更偏真人 P 图感、学校画册感和毕业纪念封面感的校园纪念图。整体应真实、干净、高级，像一张经过精修和设计排版的毕业纪念写真，而不是奇幻海报、梦幻水彩、电影概念图或过度拼贴海报。禁止魔法光、神话式云海、超现实背景、廉价 AI 海报质感。

围绕【学校名称】构建清晰但克制的学校身份层。画面必须包含：学校名称、【毕业时间 / 毕业年份】、校训或学校精神短句、学校代表色，以及至少一个经过参考图校验的学校视觉元素，例如代表建筑、校门、校徽小章、标题旁印章、页脚建筑线稿、右下角校门轮廓或校园建筑剪影。若用户没有提供毕业时间，应届毕业方向默认使用当前年份；若用户提供月份、季节或具体日期，版面允许时作为小字信息层出现，否则压缩为年份。学校建筑和关键符号必须来自网页搜索和官方 / 高可信参考图；如果无法确认，就缩小、抽象化或省略，不要编造“差不多”的学校元素。

如果提供了【姓名】，可将姓名作为低调签名、标题旁小字、纪念信息层或画面角落标注；中文名和英文名都可以。不要让姓名喧宾夺主。

如果提供了【专业 / 学院】，让版式小字、场景道具、空间线索或低风险图标与专业气质相关。
如果提供了【校园经历 / 社团经历】，将其转化为克制的记忆节点，例如图书馆、操场、社团活动、实验室、展览、毕业晚会、校园路口或常去地点。
如果提供了【个人特质 / 希望呈现的气质】，将其转化为姿态、眼神、光线和整体情绪，例如自由、坚定、温柔、热烈、理性、勇敢、开阔。

姿态不必复刻原图。可以基于多张照片重新设计更适合毕业纪念图的姿态和表情，但必须保持本人辨识度。优先选择自然、舒展、真实、适合毕业纪念的三分之二身位或半身构图；不要强行做复杂手部动作。如果手、证书、书本、花束或学位服出现，必须保证手指、手腕、手臂和握持关系自然稳定；不稳定时宁可裁掉手部或减少道具。

画面只允许一个主要人物表达。不要额外生成第二个前景真人主体。毕业氛围可以通过小尺度远景人群、毕业帽、书本、光线、校园路、建筑线稿、学校色块和文字信息层表达。

整体排版应有学校官方画册或校刊封面的秩序感：大面积留白、清晰标题区、克制学校色、准确文字区域、真实校园光线、低风险学校符号。人物是第一视觉焦点，学校元素服务于人物和毕业纪念，不要压过脸部。

## 生成后说明

生成后简要说明：

- 使用了哪个学校。
- 使用了哪些学校建筑、校徽、校训、代表色或精神关键词。
- 哪些学校元素来自参考图，哪些因风险被缩小、线稿化或省略。
- 如果用户没有提供姓名、专业、校园经历、特质或毕业时间，提醒这些信息可以让下一版更贴近本人和校园记忆。
